# SVS模型 v2.0 运行结果记录 | 2026-04-16

## 基准验证
- Podkletnov参数: 1g → 1.1%（报告2%，量级正确）
- 500mg → 1.1%（比例吻合）

## 关键发现

### RF功率效应
- 100 W/m² → 1.1%
- 1000 W/m² → 1.46%
- 10000 W/m² → 5.04%（强依赖关系！）

### 最优旋转速度
- ω_opt ≈ 5.9×10⁶ rad/s → 引力减少40%（Podkletnov的5000rpm=524 rad/s下只有1.1%）
- 旋转是关键放大杠杆

### 多层协同放大
- γ=1.5时：100层 → 127万倍基准
- γ=2.0时：100层 → 4亿倍

### 最优工作温度
- T=4K时效果最强（相干长度最大）
- 但室温超导突破将使T_c > 300K → 室温SVS成为可能

## 达到人体悬浮的路径

1. **克级验证（立即可行）**：
   - 10层YBCO + 最优RF → ~2%（Podkletnov水平）
   
2. **公斤级（1-3年）**：
   - 100层高温超导 + 最优RF + 高速旋转 → ~50%
   
3. **人体级（5-10年）**：
   - 5000层 + 室温超导 + 最优RF → ~100%悬浮
