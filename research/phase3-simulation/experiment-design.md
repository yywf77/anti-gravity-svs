# SVS效应微型验证实验设计
## 反重力衣研究 — Phase 3
## 问天计划 | 2026-04-16

---

## 一、实验目标

### 1.1 核心目标
**独立复现 Podkletnov 型 SVS 效应，检测到 >0.5% 的引力减少**

### 1.2 具体指标
| 指标 | 目标值 | 说明 |
|------|--------|------|
| 测试质量 | 1-5g 石英球 | 非磁性、低密度、高精度 |
| 灵敏度 | Δg/g < 0.1% | 比目标低一个量级 |
| 可重复性 | 3次独立测量 | 统计显著性 |
| 信噪比 | > 5σ | 明确信号 |

### 1.3 实验时间线
```
第1-2周：装置搭建与调试
第3-4周：基准测量（无RF）
第5-8周：RF参数扫描
第9-12周：旋转速度扫描 + 数据分析
第13周：撰写实验报告
```

## 二、实验装置

### 2.1 整体架构

```
[RF信号发生器]
     ↓
[功率放大器] → [RF天线] → [照射YBCO超导盘]
     ↓
[高速旋转电机] → [YBCO盘]
     ↓
[石英测试球] → [高精度扭转摆]
     ↓
[光学位移传感器] → [数据采集]
```

### 2.2 超导样品

**方案A（推荐）：YBCO商业薄膜**
- 供应商：北京有色金属研究总院 / 南京大学超导研究所
- 规格：直径25mm，厚度0.5mm，Tc=93K
- 基底：MgO 或 LaAlO₃（绝热、绝缘）
- 价格：约¥500-2000/片

**方案B：BSCCO管**
- 更高Tc（110K），但价格更贵
- 可制成环形转子

**多层制作（用于验证多层放大）**：
- 将5-10片YBCO薄膜用银胶粘合叠加
- 或使用REBCO涂层导体（上海超导科技可购）

### 2.3 旋转系统

**高速电机选型**：
| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 转速 | 0-50,000 rpm | 可调 |
| 轴承 | 磁悬浮或陶瓷球轴承 | 减少振动 |
| 轴向跳动 | < 5μm | 关键指标 |
| 真空腔 | 需低真空（~10⁻³ Torr） | 减少气流扰动 |

**推荐型号**：
- 牙科高速涡轮手机（Air Turbines B45）：可达 400,000 rpm，需要气源
- 磁悬浮高速电机（如 Newport 动态空气轴承）：价格约 ¥5-20万

**简化方案**（预算有限）：
- 使用数控雕刻机的电主轴（30,000 rpm）：约 ¥2000
- 软木塞 + 超细钢丝悬挂消除摩擦

### 2.4 RF系统

**RF参数**：
| 参数 | 扫描范围 | 步长 |
|------|---------|------|
| 频率 | 1 MHz - 100 GHz | 对数等分 |
| 功率 | 0.1 - 100 W | 10^n 等分 |
| 波形 | 连续波(CW) | - |
| 照射方式 | 环绕天线 | - |

**设备选型**：
- 信号源：HP 8640B 或同类（频率覆盖到 GHz）
- 功率放大器：Mini-Circuits ZVE-8G+（0-8 GHz，40dBm）
- RF天线：自制半波偶极子天线或波导天线

**功率密度估算**：
- 距离天线 5cm，功率 10W，天线增益 2 → 
- P ≈ 10/(4π×0.05²) ≈ 320 W/m²

### 2.5 扭转摆测量系统

**扭转摆设计**：
```
    悬挂线（Wollaston银丝，d=10μm，L=50cm）
           |
    悬浮质量（石英球，d=5mm）
           |
    反射镜（用于光学读出）
```

**参数计算**：
- 扭转弹簧系数：κ = GJ/r ≈ 10⁻⁶ N·m/rad（Wollaston丝）
- 扭转共振频率：f = 1/(2π)×√(κ/I) ≈ 0.1-1 Hz
- 位移灵敏度：光学杠杆法可达 Δx ≈ 10⁻⁸ m

**关键校准**：
- 用已知质量的微小砝码进行静态校准
- 用电磁线圈进行动态校准

## 三、实验步骤

### 3.1 装置搭建

**Step 1：超导样品安装**
1. 将YBCO盘用低温胶固定于旋转轴末端
2. 确保接触良好，旋转时无偏心
3. 用液氮预冷至77K
4. 测量Tc，确认超导转变正常

**Step 2：扭转摆安装**
1. 用Wollaston银丝悬挂石英球
2. 调节悬挂长度使扭转共振频率在 0.1-1 Hz
3. 在室温下校准扭转常数 κ
4. 安装光学反射镜

**Step 3：RF天线安装**
1. 将RF天线置于YBCO盘侧方，距离可调
2. 连接RF信号源和功率放大器
3. 用功率计校准输出功率

**Step 4：初步调试**
1. 在无旋转、无RF状态下测量扭转摆基准位置
2. 确认环境振动噪声水平
3. 调整悬挂系统消除摆动

### 3.2 基准测量（无SVS效应）

**Step 5：零场基准**
1. YBCO保持在常导态（T > Tc = 93K）
2. 旋转速度从0扫描到最大值
3. **预期结果**：无引力减少（基线）
4. 如果出现信号 → 系统误差，需要排查

**Step 6：超导态无RF基准**
1. YBCO冷却至77K，进入超导态
2. 无RF照射，旋转速度扫描
3. **预期结果**：可能有微弱引力减少（<0.1%）
4. 这部分是纯GEM效应（如果存在的话）

### 3.3 SVS效应测量

**Step 7：固定频率，扫描功率**
- RF频率固定在 f = 16 GHz（理论预测最优）
- RF功率从 0.1W 扫描到 100W
- 记录每个功率点的扭转摆偏转

**Step 8：固定功率，扫描频率**
- RF功率固定在 P = 10W
- RF频率从 1MHz 扫描到 100GHz
- 寻找共振峰

**Step 9：旋转速度扫描（最优RF条件）**
- RF固定在最优(f, P)组合
- 旋转速度从 1,000 rpm 到最大转速
- 寻找非线性临界点

**Step 10：多层放大测试**
- 在确认单层SVS效应后，叠加2层、5层、10层YBCO
- 测量是否出现N^γ放大规律

## 四、数据分析

### 4.1 数据处理流程

```
原始数据（扭转摆位移）
    ↓
扣除背景漂移
    ↓
旋转速度归一化
    ↓
扣除无超导对照数据
    ↓
引力减少 Δg/g = Δx × κ / (m × g × L)
    ↓
误差分析（统计误差 + 系统误差）
    ↓
最终结果
```

### 4.2 关键公式

扭转摆偏转角 θ 对应的引力减少：

```
Δg/g = (κ × θ) / (m × g × L)

其中：
κ = GJ/r = 扭转弹簧常数
θ = 扭转偏转角（从光学杠杆读出）
m = 测试质量
g = 9.8 m/s²
L = 悬挂长度
```

### 4.3 误差来源

| 误差来源 | 量级 | 缓解方法 |
|---------|------|---------|
| 温度漂移 | 0.1%/K | 液氮恒温 |
| 振动噪声 | 10⁻⁴ - 10⁻³ | 隔振台、信号平均 |
| 磁场力 | 未知 | 非磁性石英球 |
| 气流 | 10⁻⁴ | 真空腔或挡风罩 |
| 热辐射压力 | 10⁻⁶ | 可忽略 |
| 统计误差 | ∝ 1/√N | 长时间平均 |

## 五、安全注意事项

### 5.1 低温安全
- 液氮温度：-196°C，防冻伤
- 操作时戴防冻手套和护目镜
- 液氮杜瓦瓶需防倾倒

### 5.2 RF辐射安全
- RF功率>10W时：禁止直视天线
- 使用RF功率计监控
- 确保接地良好

### 5.3 高速旋转安全
- 转速>10,000 rpm时需要防护罩
- 旋转件需做动平衡
- 禁止在旋转时调整样品

## 六、预期结果解读

### 6.1 如果检测到>0.5%信号
→ **SVS效应得到独立验证！**
→ 可以开始设计公斤级悬浮实验
→ 准备发表预印本论文

### 6.2 如果仅检测到<0.1%信号
→ SVS效应存在但较弱
→ 可能是参数未优化（继续扫描）
→ 需要更高灵敏度的测量

### 6.3 如果未检测到信号
→ 检查系统灵敏度是否足够
→ 尝试不同的YBCO样品
→ 重新评估SVS理论模型
